---Description---

This project is a Student Grade Management System that allows users to register students and manage their grades.
The system helps organize academic records and calculate student performance.

---Features---

Feature 1 — Student Registration
Allows users to register and store student information.

Feature 2 — Subject and Grade Registration
Allows users to associate subjects and grades with each student.

Feature 3 — Information Display
Allows users to view the registered academic information.

Feature 4 — Result Analysis
Allows the system to calculate averages and identify outstanding students.
